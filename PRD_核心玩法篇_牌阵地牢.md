# 产品需求文档（PRD）- 核心玩法篇

## 《牌阵地牢》Deck Dungeon: Mini

**版本**：v1.0 MVP  
**目标平台**：iOS（iPhone/iPad）  
**定价策略**：$2.99 / ¥18（买断制+内购扩展）  
**开发周期**：3-4个月  
**团队规模**：4-6人（1策划+2程序+2美术+1音效）

---

## 1. 游戏概念

### 一句话描述核心玩法

> **"用扑克牌的规则玩Roguelike——在4×4方格中摆放卡牌，组成牌型激活技能，击败一波波怪物爬向地牢100层。"**

### 概念解析

| 维度 | 设计 | 对标参考 |
|------|------|----------|
| 核心机制 | 卡牌摆放（Deck Placement）+ 扑克牌型判定 | 《小丑牌》+《杀戮尖塔》 |
| 战斗节奏 | 单局5-10分钟，可随时暂停/继续 | 《以撒的结合》 |
| 美术风格 | 暗黑童话风，2D手绘+动态光影 | 《月圆之夜》 |
| 游戏类型 | Roguelike卡牌构筑 + 轻度放置挂机 | 独创组合 |

## 2. 核心循环

### 2.1 每分钟在做什么？

```
【一局战斗的典型节奏（5-8分钟）】

第0-30秒：观察本轮敌人 → 思考手牌摆放策略
第30秒-2分钟：拖动卡牌到4×4方格 → 组成牌型（对子/顺子/同花等）
第2-3分钟：激活牌型触发技能 → 攻击/防御/抽牌/回血
第3-5分钟：应对敌人行动 → 调整牌阵应对BOSS机制
第5-8分钟：战斗结束 → 选择奖励（新卡/金币/遗物）
```

**核心交互**：
- **拖拽摆放**：卡牌拖入4×4方格，可随时调整位置
- **牌型判定**：实时显示当前组成的扑克牌型（对子/三条/顺子/同花/葫芦/四条/同花顺）
- **连击系统**：连续组成高级牌型触发额外伤害

### 2.2 每小时在做什么？

```
【放置挂机收益】

离线期间：
└─ 已通关层数会累积"地牢灵气"（每小时自动获取）
└─ 灵气可用于：解锁新卡牌 / 升级卡牌 / 购买皮肤

上线后：
└─ 领取离线收益（最多累积8小时，防过度挂机）
└─ 进行1-2局地牢挑战（每次5-10分钟）
└─ 调整牌库构筑（尝试新套路）
```

### 2.3 每天在做什么？

```
【每日循环（20-30分钟/天）】

每日任务（必做，5分钟）：
└─ 完成1次"每日挑战"（固定种子，全服统一）
└─ 领取登录奖励（连续7天有额外奖励）

核心玩法（可选，15-20分钟）：
└─ 挑战更高层数（冲层）
└─ 尝试新牌组Build（构筑乐趣）
└─ 完成成就目标（收集向）

周末特别活动：
└─ 周六/周日开启"无尽模式"（比谁能走得更远）
```

| | 战斗特效 | P1 | 简洁有力的2D动画 |

| | 卡牌升级 | P0 | 铜→银→金，改变数值+效果 |
| | 牌组编辑 | P1 | 最多20张卡组的构筑界面 |
| **养成系统** | 遗物系统（15个） | P1 | 永久被动加成，每次冒险可选3个 |

| **地牢模式** | 标准模式（100层） | P0 | 主玩法，分10个区域 |

| | 无尽模式 | P2 | 周末开放，测试极限 |

| | 快速挂机 | P1 | 看广告/付费获取2小时收益 |
| **商店** | 卡牌解锁包 | P2 | 扩展包1：+15张新卡（$0.99） |
| | 皮肤商店 | P2 | 卡背/角色皮肤/特效 |
| **系统** | 新手引导（5分钟） | P0 | 强制教程，必须完成 |
| | 成就系统（20个） | P1 | 引导玩家尝试不同套路 |
| | 云存档 | P1 | iCloud同步 |
| | 多语言（简中/繁中/英/日） | P1 | 付费榜必须国际化 |

### 明确不做（保持小体量）

- ❌ 联网对战（开发成本高，匹配难）
- ❌ 复杂剧情（Roguelike不需要）
- ❌ 装备/道具系统（已简化为遗物）
- ❌ 公会/好友系统（保持单机体验）
- ❌ 抽卡系统（买断制游戏，避免付费墙）

## 4. 付费点设计

### 4.1 为什么玩家愿意付费？

| 付费动机 | 设计支撑 | 价格定位 |
|----------|----------|----------|
| **支持开发者** | 游戏品质过硬，买断制无广告 | $2.99基础价 |
| **扩展内容** | 新卡牌包增加玩法深度 | $0.99/包（每季度1包） |
| **个性化** | 精美皮肤满足收藏欲 | $0.99-1.99/套 |
| **省时间** | 快速挂机跳过等待 | ¥6/次 or 看广告免费 |

### 4.2 付费转化率优化策略

```
【转化率漏斗】

曝光 → 兴趣 → 试用 → 付费 → 复购
  │      │      │      │      │
  │      │      │      │      └─ 新扩展包推送
  │      │      │      └─ 首充双倍+限时折扣
  │      │      └─ 免费试玩前20层
  │      └─ 商店页面展示核心卖点
  └─ ASO优化+游戏媒体评测
```

**具体策略**：

1. **试玩机制**：免费下载，第20层后弹窗付费解锁完整版
   - 转化率预估：8-12%（参考《纪念碑谷》模式）

2. **首充设计**：解锁完整版后，首单扩展包半价
   - 刺激复购，ARPPU预估$4-6

3. **限时活动**：节日限定皮肤（错过等一年）
   - 制造稀缺感，提升付费紧迫感

4. **无广告承诺**：买断制无强制广告，快速挂机可选择看广告
   - 尊重玩家选择权，提升口碑

### 4.3 收入预估模型

```
保守预估（首年）：
- 下载量：50万（付费榜曝光+口碑传播）
- 付费转化：10% = 5万付费用户
- 基础收入：5万 × $2.99 = $14.95万
- 扩展包购买率：30% = 1.5万 × $0.99 = $1.49万
- 皮肤/快速挂机：$1万
- 首年总计：约$17-20万（约120-140万人民币）

乐观预估（进入付费榜Top10）：
- 下载量：200万
- 收入：$70-100万（约500-700万人民币）
```

## 5. 留存设计

### 5.1 第7天还在玩的理由

| 留存节点 | 设计策略 | 具体机制 |
|----------|----------|----------|
| **Day 1** | 快速上手 | 5分钟教程后立即可玩，第1局即爽 |
| **Day 2** | 解锁成就 | 完成"首次通关10层"解锁新角色 |
| **Day 3** | 构筑乐趣 | 解锁牌组编辑，开始研究Build搭配 |
| **Day 4** | 社交比较 | 每日挑战排行榜，看到好友/全球排名 |
| **Day 5** | 深度内容 | 解锁天赋树，开始规划长期成长 |
| **Day 6** | 新鲜挑战 | 首次遇到隐藏BOSS，激发探索欲 |
| **Day 7** | 持续奖励 | 7日登录送限定卡背，养成习惯 |

### 5.2 第30天还在玩的理由

```
【长线留存机制】

1. Build深度（核心）
   └─ 30张卡 × 3种升级 × 5个角色 = 数百种组合
   └─ 遗物搭配进一步扩展可能性
   └─ 每次冒险都是新的解谜体验

2. 冲层挑战（目标）
   └─ 100层通关 → 解锁困难模式（层数上限200）
   └─ 困难通关 → 解锁噩梦模式（层数上限300）
   └─ 每个模式有独立排行榜

3. 收集养成（成就）
   └─ 卡牌全收集（30→60→90张）
   └─ 遗物全解锁（15→30→45个）
   └─ 成就系统（100+个目标）
   └─ 图鉴100%完成度

4. 每日新鲜（习惯）
   └─ 每日挑战：固定种子的公平竞技
   └─ 无尽模式：周末开放的极限挑战
   └─ 赛季系统：每月重置的排名赛季
```

**30天留存目标**：25-30%（参考《杀戮尖塔》移动端数据）

## 6. 社交元素

### 6.1 是否需要社交？

**结论**：需要轻度社交，但**严格保持单机体验的核心地位**。

理由：
- ✅ 社交是留存的重要驱动力（攀比+炫耀）
- ✅ 排行榜带来重玩价值
- ❌ 强制社交会赶走核心单机玩家
- ❌ 联机对战开发成本过高

### 6.2 如何设计不让单机玩家反感？

| 功能 | 设计 | 单机玩家体验 |
|------|------|--------------|
| **全球排行榜** | 仅显示Top100+自己的排名 | 可选择不看，无推送打扰 |
| **每日挑战** | 统一种子，纯技术比拼 | 本质是"异步PVP"，无实时交互 |
| **幽灵对战** | 可选择观看高排名玩家的该局录像 | 学习向，非强制 |
| **成就分享** | 通关/高分时可选择分享到社交媒体 | 纯自愿，无奖励驱动 |
| **好友排行榜** | 可选同步Game Center好友 | 不强制，可随时关闭 |

**明确不做**：
- ❌ 好友助力/赠送体力
- ❌ 公会系统
- ❌ 聊天功能
- ❌ 组队副本

**设计原则**：社交是"展示台"，不是"必需品"。

## 7. 差异化

### 7.1 竞品分析

| 竞品 | 优点 | 缺点 | 我们的机会 |
|------|------|------|------------|
| **《小丑牌》** | 创新的扑克牌型机制 | 纯单机无放置、后期内容少 | 加入放置元素，长线运营 |
| **《杀戮尖塔》** | 卡牌构筑经典 | 单局过长（30分钟+）、上手门槛高 | 单局5分钟、极简规则 |
| **《月圆之夜》** | 精美美术、剧情驱动 | 构筑深度相对浅 | 更深策略+无剧情干扰 |
| **《幸运房东》** | 放置+肉鸽结合 | 美术简陋、受众有限 | 更高品质美术+移动端优化 |

### 7.2 我们独特在哪？

```
【核心差异化：4×4牌阵系统】

传统Roguelike卡牌：
└─ 手牌打出 → 进入弃牌堆 → 抽新牌
└─ 决策点：每回合选1-3张牌打出

我们的创新：
└─ 手牌摆放 → 组成牌型 → 激活对应技能
└─ 决策点：卡牌位置影响多个牌型组合

举例：
┌────┬────┬────┬────┐
│ A♠ │ 2♥ │ 3♦ │ 4♣ │  ← 构成顺子：触发连击
├────┼────┼────┼────┤
│ A♥ │ 5♠ │ 6♦ │ 7♥ │  ← A♠+A♥构成对子：触发加倍
├────┼────┼────┼────┤
│ 8♣ │ 9♠ │ 10♥│ J♦│  ← 同花色聚集：触发同花
├────┼────┼────┼────┤
│ Q♣ │ K♠ │ ?  │ ?  │  ← 空格等待新牌
└────┴────┴────┴────┘

优势：
1. 空间策略 + 卡牌构筑 = 全新体验
2. 视觉直观，比纯数字更易理解
3. 每局不同，布局千变万化
```

### 7.3 一句话差异化定位

> **"《小丑牌》的空间策略版——用摆放代替打出，每一局都是新的拼图。"**

### 7.4 营销卖点

| 卖点 | 传播话术 |
|------|----------|
| 轻量精品 | "地铁上就能通关一局的地牢冒险" |
| 策略深度 | "30张卡，300种Build，10000种解法" |
| 独特机制 | "不用打牌，只用摆牌" |
| 买断制良心 | "一次付费，终身畅玩，无体力无广告" |

## 8. 风险评估与应对

| 风险 | 可能性 | 影响 | 应对措施 |
|------|--------|------|----------|
| 核心玩法不好玩 | 中 | 致命 | 开发原型阶段就做2周玩家测试 |
| 付费转化率低 | 中 | 高 | 前20层免费试玩，降低决策门槛 |
| 竞品提前上线 | 低 | 中 | 加快开发节奏，争取3个月内首测 |
| 美术风格不受欢迎 | 低 | 中 | 准备2-3套风格方案AB测试 |
| 后期内容不足 | 中 | 高 | MVP只做到50层，根据数据决定扩展 |
