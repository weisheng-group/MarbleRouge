# Marble Rogue - Design Document

**Project:** Marble Rogue - 弹珠肉鸽游戏  
**Date:** 2026-03-14  
**Platform:** iOS (SwiftUI + SpriteKit)  
**Pricing:** $1.99 (MVP) / $4.99 (Full)

---

## 1. Core Vision

**一句话描述：** 以撒风格的弹幕组合 + 弹珠台物理 + 肉鸽升级

**核心爽点：**
- 单指操作，秒上手
- 弹珠组合产生爆炸性视觉效果
- 每局不同Build，重玩价值高

---

## 2. Core Loop

```
瞄准 → 发射 → 弹跳得分 → 过关 → 3选1升级 → 下一关
```

### 2.1 基础操作
- **按住屏幕：** 显示瞄准线，拖动调整角度
- **松开手指：** 发射弹珠
- **弹珠静止后：** 回合结束，计算得分

### 2.2 过关条件
- 达到目标分数即可过关
- 剩余弹珠数 = 奖励分数
- 连击（单次发射多次命中）= 额外分数

---

## 3. Ball Types (10种基础弹珠)

灵感来源于《以撒的结合》经典道具：

| 编号 | 名称 | 图标 | 效果描述 |
|------|------|------|----------|
| 1 | **基础弹珠** | ● | 标准弹珠，无特殊效果 |
| 2 | **分裂弹珠** | ⚫⚫ | 击中后分裂成2个小弹珠 |
| 3 | **炸弹弹珠** | 💣 | 击中后2秒延迟爆炸 |
| 4 | **追踪弹珠** | 🎯 | 小幅自动修正弹道追踪目标 |
| 5 | **穿透弹珠** | ➡️ | 穿透目标继续飞行 |
| 6 | **橡胶弹珠** | 🔄 | 墙壁弹射5次，每次伤害+50% |
| 7 | **导弹弹珠** | 🚀 | 发射后自动追踪最近目标 |
| 8 | **环形弹珠** | ⭕ | 蓄力发射，环形扩散 |
| 9 | **激光弹珠** | ⚡ | 发射持续激光束，穿透一切 |
| 10 | **四重弹珠** | ❖ | 同时发射4颗扇形弹珠 |

### 3.1 稀有度分级

- **普通 (White):** 基础、分裂、穿透、橡胶
- **稀有 (Blue):** 炸弹、追踪、导弹
- **史诗 (Purple):** 环形、激光、四重
- **传说 (Gold):** 全屏清除 (仅第20关后出现)

### 3.2 叠加规则

- 同类升级最多叠 **3层**
- 分裂×3 = 分裂成 **8个** 弹珠 (2^3)
- 四重×3 = 每次发射 **12颗** 弹珠 (4×3)
- 不同类效果 **全部叠加**

## 4. Combo System (组合系统)

### 4.1 经典组合

| 组合A | 组合B | 结果效果 | 命名 |
|-------|-------|----------|------|
| 追踪 | 分裂 | 小弹珠也带追踪 | 蜂群 |
| 导弹 | 分裂 | 命中后分裂成追踪小导弹 | 子母弹 |
| 激光 | 橡胶 | 激光在墙壁间反射 | 镭射网格 |
| 环形 | 追踪 | 旋转弹珠自动追踪飞出 | 卫星打击 |
| 四重 | 炸弹 | 4颗炸弹同时轰炸 | 地毯轰炸 |
| 穿透 | 追踪 | 穿透过程中持续追踪 | 穿心箭 |
| 橡胶 | 分裂 | 每次弹射都分裂 | 混沌弹跳 |

### 4.2 终极组合示例

**Build: "毁灭日"**
- 四重 ×3 + 导弹 ×3 + 分裂 ×3
- 效果：12颗导弹，每颗命中分裂成8个追踪小导弹
- 视觉：满屏追踪导弹，清场效果

## 5. Level Design

### 5.1 关卡结构

- 每 **5关** 为一个阶段
- 第5关为 **Boss关** (固定靶+移动靶组合)
- 第20关后解锁 **传说升级**

### 5.2 关卡元素

| 元素 | 描述 |
|------|------|
| 钉子 (Peg) | 固定位置，弹珠碰撞后反弹 |
| 目标靶 (Target) | 击中得分，不同颜色不同分值 |
| 移动靶 (Moving) | 沿轨道移动，击中额外分数 |
| 障碍物 (Obstacle) | 阻挡弹珠，需绕行 |
| 传送门 (Portal) | 弹珠进入后从另一端传出 |
| 得分倍率区 | 弹珠经过时当前得分×2 |

### 5.3 难度曲线

- 第1-5关：静态布局，教学
- 第6-10关：加入移动靶
- 第11-15关：加入障碍物
- 第16-20关：综合挑战
- 第20关+：无限模式，难度递增

## 6. UI/UX Design

### 6.1 屏幕布局 (竖屏)

```
┌─────────────────┐
│  [分数]  [关卡]  │  ← 顶部HUD
├─────────────────┤
│                 │
│    游戏区域      │  ← 弹珠台区域
│   (占屏幕70%)   │
│                 │
├─────────────────┤
│  [瞄准线]        │  ← 底部发射区
│     👆          │
└─────────────────┘
```

### 6.2 色彩方案 (霓虹赛博朋克)

| 元素 | 颜色 | 色值 |
|------|------|------|
| 背景 | 深紫黑 | #0D0D1A |
| 弹珠 | 亮青 | #00FFFF |
| 钉子 | 霓虹粉 | #FF00FF |
| 目标 | 金黄 | #FFD700 |
| 特效 | 渐变彩虹 | - |

### 6.3 特效要求

- **击中粒子：** 击中时迸发彩色粒子
- **拖尾效果：** 弹珠飞行带渐变拖尾
- **升级提示：** 获得升级时全屏闪烁
- **连击数字：** 连击时显示浮动数字

## 7. Audio Design

### 7.1 音效

- 发射："嗖" ( pitch随蓄力时间变化 )
- 击中钉子："叮" ( 清脆金属声 )
- 击中目标："啪" ( 爽快打击感 )
- 升级选择："升级感"音效
- 过关：胜利号角

### 7.2 BGM

- 主菜单：电子合成器，舒缓
- 游戏内：节奏感强，随关卡加快
- Boss战：紧张激烈
