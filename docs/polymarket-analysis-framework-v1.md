## 一、市场筛选框架（哪些值得看）

### 1.1 筛选标准（按优先级）

| 优先级 | 标准 | 说明 |
|--------|------|------|
| **P0** | 流动性充足 | 日交易量 > $10,000（确保能进出） |
| **P0** | 你懂的事件领域 | 政治/体育/Crypto（有信息优势） |
| **P1** | 时间窗口合理 | 1-4周内出结果（避免资金长期锁定） |
| **P1** | 信息可验证 | 有明确信息源可追踪（不是黑箱） |
| **P2** | 定价明显偏离 | 你的判断 vs 市场价格差异 > 15% |

### 1.2 红灯市场（避开）

- ❌ 流动性极低（买卖价差 > 5%）
- ❌ 事件结果可被操控（内部人控制）
- ❌ 结果定义模糊（"拜登表现好"——怎么算好？）
- ❌ 时间不确定（"今年会降息吗"——可能拖到12月）

## 三、分析决策流程（标准化步骤）

### Step 1：概率评估（你判断）
```
基于我的信息，我认为该事件发生的真实概率是：___%
```

### Step 2：价格对比（计算）
```
市场价格（隐含概率）：___%
我的评估概率：___%
差异：___%

判断：
□ 市场低估（我的概率 > 市场价格）→ 买 Yes
□ 市场高估（我的概率 < 市场价格）→ 买 No
□ 定价合理 → 观望
```

### Step 3：期望值计算（EV）
```
如果买 Yes：
- 投入：$___
- 若赢：回报 $___（赔率 = 1/价格）
- 若输：损失 $___
- 预期收益 = (胜率 × 回报) - (败率 × 损失)

EV > 0 才考虑下注
```

### Step 4：仓位决策（凯利公式简化版）
```
资金分配比例 = (胜率 × 赔率 - 败率) / 赔率

简化规则：
- 高信心（>70%把握）：投入本金 10-20%
- 中等信心（50-70%）：投入本金 5-10%
- 低信心（<50%）：观望或小额试水 1-2%

单市场最大仓位：≤ 本金的 20%
单日最大亏损：≤ 本金的 10%
```

## 四、风险管理规则（铁律）

### 4.1 绝对红线

| 规则 | 说明 |
|------|------|
| **永不All in** | 单次下注 ≤ 20% 本金 |
| **止损线** | 单一持仓浮亏 -30% 强制止损 |
| **止盈线** | 盈利 +50% 考虑部分止盈 |
| **时间止损** | 临近到期无把握，提前退出 |
| **冷静期** | 连续亏损2次后，停止24小时 |

### 4.2 投资组合管理

```
同时持仓 ≤ 5 个市场
相关性分散：不要把所有押注放在同一类事件
（例如：不要同时押注多个美国大选相关市场）
```

### 4.3 资金曲线目标

| 阶段 | 本金 | 目标 | 风险等级 |
|------|------|------|----------|
| 试水期 | $100 | 熟悉流程，不亏为主 | 极低 |
| 学习期 | $100-500 | 摸索优势领域 | 低 |
| 稳定期 | $500-2000 | 正收益，控制回撤 | 中 |
| 扩张期 | $2000+ | 系统化套利 | 中高 |
